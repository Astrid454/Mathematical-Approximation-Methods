# Mathematical Approximation Methods

**Technologies:** MATLAB, Java

**Description:**  
Implementation of various numerical algorithms for mathematical approximation, including Polynomial Interpolation, Numerical Integration, and Thomas Algorithm.  

**How it works:**  
1. Initial development and algorithm testing in **MATLAB**.  
2. Further implementation in **Java** for:  
   - Graphical representation of input data  
   - Interactive use of selected numerical methods  

**Features:**  
- Choose between multiple numerical methods for approximation  
- Visualize results graphically  
- Easily extendable for additional algorithms
